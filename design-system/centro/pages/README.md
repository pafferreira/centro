# Páginas específicas do Design System

Crie arquivos `.md` neste diretório para sobrescrever regras do MASTER para páginas específicas.

## Formato

Cada arquivo deve ser nomeado como `[nome-da-pagina].md` e pode conter:

- Cores adicionais ou substitutas
- Tipografia específica
- Componentes com variações
- Layout diferenciado

## Exemplo

```
design-system/centro/pages/dashboard.md
design-system/centro/pages/room-assembly.md
design-system/centro/pages/passe-registration.md
```

## Regra de precedência

Página específica > MASTER > Defaults do TailwindCSS
